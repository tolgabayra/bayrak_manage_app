# bayrak_manage_app



Install the dependencies and devDependencies and start the server.
## Tech
- TypeScript
- ExpressJS
- TypeORM
- POSTGRESQL
- JWT

## apiserver

```sh

- cd apiserver/v1
- npm install
- create your .env
- npm start

```


## client
Install the dependencies and devDependencies and start the client next server.

## Tech
- TypeScript
- NextJS
- Axios
- TailwindCSS
- Chakra UI


```sh

- cd client
- npm install
- npm run dev

```
